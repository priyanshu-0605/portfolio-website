Vodala Priyanshu | Personal Portfolio
A premium, high-performance personal portfolio built with React and Framer Motion. This project showcases my journey as a developer, specializing in Generative AI, Data Analysis, and Modern Web Development.

Philosophy: 
"Crimson Glass"The site is built with a custom Red & Glossy aesthetic, designed to provide a high-end, immersive user experience.
Glassmorphism: Frosted glass UI components with dynamic backdrops.
Neon Accents: Strategic use of #ff0033 for high-impact visual hierarchy.
Motion Design: Powered by Framer Motion for fluid, physics-based transitions.
Performance: Optimized with Vite for near-instant load times.

Key Features
FeatureDescription
Dynamic Hero: Animated introduction with sleek entry transitions.
Interactive Timeline: A visual journey through my education and professional growth.
Project Showcase: Deep dives into projects like Salary Estimation and Flappy Bird AI.
Skills Visualizer: Modern progress bars highlighting proficiency in Python, SQL, and Gen AI.
Smart Contact: Quick-action social links and a "Click-to-Copy" phone feature.
